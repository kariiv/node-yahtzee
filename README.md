# Yahtzee SocketIO
Yahtzeee game using Socket.IO and Express

## Veebirakendused VI praktikum
Abipakett aine läbimiseks.

## Features
- Create public game
- Available game list
- Create private game with PIN code
- Animations

USING COOKIES 🍪🍪🍪
Use different browsers to play the game or incognito.

Design: kariiv & vertam
Logic: kariiv
